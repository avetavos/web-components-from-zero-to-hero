## constructor

`constructor` คือจุดที่คุณตั้งค่า internal state เริ่มต้นของ element เช่น กำหนดค่า property หรือสร้าง Shadow DOM ข้อกำหนดที่สำคัญมีดังนี้:

- ต้องเรียก `super()` เป็น statement แรกเสมอ
- **ห้าม** อ่านหรือเขียน attribute ของ element ใน constructor
- **ห้าม** แตะ children หรือ parent node ของ element ใน constructor
- **ห้าม** render เนื้อหาลงใน DOM ใน constructor

เหตุผลคือเมื่อ `constructor` ถูกเรียก element ยังไม่ได้อยู่ใน document และ attribute ยังไม่ได้ถูกกำหนดค่า

```js
class MyElement extends HTMLElement {
  constructor() {
    super(); // ต้องมาก่อนเสมอ
    this._count = 0; // ตั้งค่า internal state ได้
    this._data = null;
    // ห้ามทำ: this.getAttribute('foo') หรือ this.innerHTML = '...'
  }
}
```

## connectedCallback

`connectedCallback` จะถูกเรียกทุกครั้งที่ element ถูกแทรกเข้าไปใน document ที่เชื่อมต่ออยู่ นี่คือจุดที่ถูกต้องและปลอดภัยสำหรับ:

- การ render เนื้อหา HTML ของ element
- การอ่าน attribute เพื่อกำหนดพฤติกรรม
- การลงทะเบียน event listener
- การเริ่มต้น timer หรือ fetch ข้อมูล

สิ่งที่ต้องระวังคือ `connectedCallback` จะถูกเรียกซ้ำ**ทุกครั้ง**ที่ element ถูกแทรกเข้าไปใหม่ ถ้าย้าย element ไปยัง parent อื่นด้วย `appendChild` callback นี้จะถูกเรียกอีกครั้ง

```js
class MyElement extends HTMLElement {
  connectedCallback() {
    // อ่าน attribute ได้อย่างปลอดภัย
    var label = this.getAttribute('label') || 'default';
    this.textContent = label;
    // เพิ่ม listener ได้
    this.addEventListener('click', this._handleClick);
  }
}
```

## ตัวอย่างที่รันได้จริง

export const ctorHtml = `<div id="log" style="font-family:monospace;font-size:.85rem;background:#f3f4f6;padding:.5rem;border-radius:4px;margin-bottom:.5rem;min-height:2rem"></div>
<ctor-demo greeting="Hello"></ctor-demo>`;

export const ctorJs = `class CtorDemo extends HTMLElement {
  constructor() {
    super();
    this._count = 0;
    var log = document.getElementById('log');
    if (log) { var p = document.createElement('p'); p.style.margin = '0'; p.textContent = '> constructor called (no DOM work here)'; log.appendChild(p); }
  }
  connectedCallback() {
    this._count = this._count + 1;
    var greeting = this.getAttribute('greeting') || 'Hi';
    this.style.display = 'block';
    this.style.padding = '.5rem 1rem';
    this.style.background = '#d1fae5';
    this.style.borderRadius = '6px';
    this.style.marginTop = '.5rem';
    this.textContent = greeting + '! Connected #' + this._count;
    var log = document.getElementById('log');
    if (log) { var p = document.createElement('p'); p.style.margin = '0'; p.textContent = '> connectedCallback called (count: ' + this._count + ')'; log.appendChild(p); }
  }
}
customElements.define('ctor-demo', CtorDemo);`;

<LivePreview client:visible html={ctorHtml} js={ctorJs} />

สังเกตว่า `constructor` ถูกเรียกก่อน แต่ไม่ได้แตะ DOM ส่วน `connectedCallback` จะนับ `_count` ขึ้นทุกครั้งที่ element ถูกแทรกเข้าไปใหม่

<Tip title="ต้องเรียก super() ก่อนเสมอ">
บรรทัดแรกของ constructor ใน custom element ทุกตัวต้องเป็น `super()` เสมอ การเรียก `super()` จะเรียก constructor ของ `HTMLElement` และตั้งค่า internal state ของ element หากละเว้น `super()` จะเกิด `ReferenceError`
</Tip>
