  const { LitElement, html, css } = await import('https://esm.sh/lit');
  class LitCard extends LitElement {
    static properties = { label: { type: String }, count: { type: Number } };
    static styles = css\`
      :host { display: block; font-family: sans-serif; background: #f3f3f3; border-radius: 8px; padding: 12px 16px; max-width: 200px; }
      h3 { margin: 0 0 8px; color: #333; }
      button { background: #6200ee; color: #fff; border: none; border-radius: 4px; padding: 6px 14px; cursor: pointer; margin: 2px; }
      span.val { font-size: 1.4rem; font-weight: bold; color: #6200ee; display: block; margin: 8px 0; }
    \`;
    constructor() { super(); this.label = ''; this.count = 0; }
    render() {
      return html\`
        <h3>\${this.label}</h3>
        <span class="val">\${this.count}</span>
        <button @click=\${() => this.count++}>+1</button>
        <button @click=\${() => this.count = 0}>Reset</button>
      \`;
    }
  }
  customElements.define('lit-card', LitCard);
})();`;

## LitElement คืออะไร

`LitElement` คือ base class ที่ extend จาก `HTMLElement` มาตรฐาน ซึ่งให้สามสิ่งหลัก:

1. **Reactive properties** — ประกาศใน `static properties` และ Lit จะ watch การเปลี่ยนแปลงเพื่อ schedule re-render
2. **Declarative render** — เมธอด `render()` คืน `TemplateResult` จาก `html` tagged template literal
3. **Scoped styles** — ประกาศใน `static styles` ด้วย `css` tagged template literal และ Lit จะนำเข้าสู่ Shadow DOM

## Reactive Properties

การประกาศ property ใน `static properties` ทำให้ Lit รู้ว่าต้อง watch property นั้น:

```js
static properties = {
  label: { type: String },
  count: { type: Number },
};
```

เมื่อ `this.count` เปลี่ยนค่า Lit จะ schedule microtask update และเรียก `render()` โดยอัตโนมัติ

## เมธอด render()

`render()` ต้องคืน `TemplateResult` ที่สร้างด้วย `html` tagged template literal:

```js
render() {
  return html`
    <h3>${this.label}</h3>
    <span>${this.count}</span>
    <button @click=${() => this.count++}>+1</button>
  `;
}
```

Lit จะ diff `TemplateResult` นี้กับ DOM ที่มีอยู่ และอัปเดตเฉพาะส่วนที่เปลี่ยนแปลง

## Scoped Styles

ใช้ `css` tagged template literal ใน `static styles` เพื่อกำหนด styles:

```js
static styles = css`
  :host { display: block; padding: 12px; }
  button { background: #6200ee; color: #fff; }
`;
```

Lit จะนำ stylesheet นี้เข้าสู่ Shadow DOM ของ element ทำให้ styles ไม่รั่วออกไปหรือถูกรบกวนจากภายนอก

<Tip title="Properties vs attributes">
Lit ซิงค์ property (`this.count`) กับ attribute (`count="0"`) โดยอัตโนมัติเมื่อตั้ง `reflect: true` ในออปชันของ property โดยไม่มี `reflect` property จะเป็นแหล่งความจริงหลัก และ attribute จะถูกใช้เฉพาะตอนเริ่มต้นเท่านั้น
</Tip>
