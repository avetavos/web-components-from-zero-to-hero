## Named slots

**named slot** คือ `<slot>` element ที่มี `name` attribute อยู่ใน shadow tree จับคู่กับ `slot` attribute บนลูกใน light DOM browser จะจับคู่ทั้งสองด้วยชื่อ และ render ลูกแต่ละตัวจาก light DOM ที่จุดแทรกที่ตรงกัน การใช้ named slot หลายตัวช่วยให้คอมโพเนนต์ตัวเดียวเปิดเผยพื้นที่เนื้อหาที่แตกต่างกันได้หลายส่วน ซึ่งเป็นแพทเทิร์นที่พบบ่อยในคอมโพเนนต์ประเภท card, dialog และ layout

```html
<my-card>
  <span slot="header">Card Title</span>
  <p slot="body">Card body text here.</p>
</my-card>
```

```js
class MyCard extends HTMLElement {
  connectedCallback() {
    const shadow = this.attachShadow({ mode: 'open' });
    shadow.innerHTML = `
      <style>...</style>
      <div class="card">
        <header><slot name="header">Untitled</slot></header>
        <section><slot name="body">No body.</slot></section>
      </div>
    `;
  }
}
customElements.define('my-card', MyCard);
```

## slot attribute ที่ไม่ตรงกัน

ลูกใน light DOM ที่มี `slot` attribute ซึ่ง **ไม่** ตรงกับ named slot ใด ๆ ใน shadow tree จะถูกข้ามไปอย่างเงียบ ๆ คือไม่ถูก render ที่ไหนเลย นั่นหมายความว่าการพิมพ์ชื่อ slot ผิดจะทำให้เนื้อหามองไม่เห็น ดังนั้นจึงควรตรวจสอบชื่อ slot ให้ดีเมื่อ debug เนื้อหาที่หายไป

## default และ named พร้อมกัน

element หนึ่งสามารถมีทั้ง named slot และ default slot พร้อมกันได้ ลูกที่ตรงกับ named slot จะถูกกระจายไปยัง named slot ที่ตรงกัน ส่วนลูกที่เหลือซึ่งไม่มี `slot` attribute จะตกลงไปอยู่ใน default slot สิ่งนี้ทำให้สร้างคอมโพเนนต์ที่มีส่วนที่มีโครงสร้าง (header, footer) บวกกับพื้นที่เนื้อหาแบบอิสระได้อย่างง่ายดาย

export const namedSlotsHtml = `<my-card>
  <span slot="header">My Card Title</span>
  <p slot="body">This is the card body. Named slots let you define multiple insertion points.</p>
</my-card>
<my-card>
  <span slot="header">Fallback demo</span>
</my-card>`;

export const namedSlotsJs = `class MyCard extends HTMLElement {
  connectedCallback() {
    var shadow = this.attachShadow({ mode: 'open' });
    shadow.innerHTML = '<style>:host{display:block;font-family:sans-serif;margin:.5rem 0}'+
      '.card{border:2px solid #6366f1;border-radius:8px;overflow:hidden}'+
      'header{background:#6366f1;color:#fff;padding:.5rem 1rem;font-weight:bold}'+
      'section{padding:.75rem 1rem;color:#333}'+
      'em{color:#aaa;font-style:italic}</style>'+
      '<div class="card">'+
      '<header><slot name="header">Untitled Card</slot></header>'+
      '<section><slot name="body"><em>No body provided.</em></slot></section>'+
      '</div>';
  }
}
customElements.define('my-card', MyCard);`;
